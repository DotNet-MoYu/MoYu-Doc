# 说明
MoYu框架文档,项目地址: [https://gitee.com/dotnetmoyu/MoYu](https://gitee.com/dotnetmoyu/MoYu)
本文档基于 [VitePress](https://vitepress.dev/zh/) 构建,文档基本框架已经搭建完毕,后续会逐步完善文档内容。
想要贡献文档内容，只需要在对应的文件夹下编写对应的`markdown`文件即可。
### 本地运行

```bash
# 安装依赖
npm install

# 本地浏览
npm run docs:dev 
```

### 发布部署

```bash
npm run docs:build 
```
